See Twee docs at https://github.com/tweecode/twee/blob/master/README.markdown
(Though note that the current working version of twee is actually contained inside the twine repository at https://github.com/tweecode/twine ). 

To compile twee source file on Windows using Python 2.x, with SugarCube2 target:

Open Windows terminal to current directory (shift right-click in File Explorer "open command window here") and type:

python .\twine\twee -t sugarcube-2 main.twee > .\build\story.html
