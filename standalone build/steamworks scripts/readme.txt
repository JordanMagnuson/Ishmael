This folder contains the steamworks sdk scripts that are used by SteamPipe to package and upload the game for steam.

-------------------
Quickhelp guide:
-------------------
Download the latest steamworks sdk from: https://partner.steamgames.com/downloads/steamworks_sdk.zip

Run \sdk\tools\ContentBuilder\builder\steamcmd.exe

login by typing "login [username]" then entering password when prompted

Then to build and upload the game:
	Steam>run_app_build "C:\Users\Jordan\Google Drive\Twine\Ishmael (Forever)\standalone build\steamworks scripts\app_build_981310.vdf"
